Pré-requisitos e Instalação

Antes de executar o projeto, você precisa instalar as bibliotecas externas do Python utilizadas para a geração de gráficos, relatórios e manipulação de imagens.

Abra o seu terminal na pasta do projeto e execute o seguinte comando:

```bash
pip install pandas matplotlib pillow

Após instalar as dependências acima, inicie a aplicação com o comando:

```bash
python main.py
